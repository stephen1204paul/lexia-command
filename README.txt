== Description ==

Plugin to supercharge your WordPress workflow.

Carry out your most common tasks with a few keystrokes.

Use the command bar to quickly navigate your site, search for content, and execute administrative tasks.

Press `Ctrl+K` (Cmd+K on Mac) to open the command bar.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Search for `lexia-command` in the WordPress plugins dashboard
2. Install the plugin
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy!
